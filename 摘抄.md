**父类容器的 z-index 优于子类 z-index**


``` css
<style>
	/* clearfix */
	.clearfix:after {
	   content: "."; /* Older browser do not support empty content */
	   visibility: hidden;
	   display: block;
	   height: 0;
	   clear: both;
	}
	.clearfix {zoom: 1;} /* 针对 IE 不支持 :after */
</style>
```
